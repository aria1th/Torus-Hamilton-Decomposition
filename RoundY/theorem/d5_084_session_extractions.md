# D5 084 Session Extractions

This note records the small subset of old `note_s*.md` and `codex_work_s*.md`
files that still contain useful mathematical or compute-side content worth
remembering.

It is **not** a replacement for the canonical `076–083` theorem chain.
Its purpose is narrower:

1. identify which session notes still carry a real idea or reduction;
2. say whether that idea is already promoted elsewhere;
3. make sure those insights are not lost in the session-log pile.

## 1. Main conclusion

Most `note_s*.md` and `codex_work_s*.md` files are historical search/task logs.
They should not be treated as theorem references.

But there is a small cluster of genuinely useful session notes from the
`041–055` compression stage and the `045–052` carry/tau stage.

Those are the ones worth remembering.
