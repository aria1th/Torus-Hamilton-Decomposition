# D5 073 Static Decorated Bridge Obstruction

This note gives a stable descriptive home for the obstruction result originally
recorded in:

- `theorem/d5_073_r1.md`

## Statement

The naive chain-static decorated bridge `(beta,a,b)` is exact on each fixed
chain but does **not** globalize as a deterministic quotient across splices.

So the live bridge target cannot be merely a static decorated chain law. It has
to be a splice-compatible dynamic bridge.

## Why this matters

This is the precise point where the project learned that:

- the bare bridge `beta` is too coarse for `epsilon4`,
- but static decoration is still not enough globally.

That obstruction is the bridge from the early decorated notes to the later
dynamic boundary odometer `(beta,q,sigma)` / `(beta,delta)`.
