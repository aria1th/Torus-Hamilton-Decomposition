\paragraph{Universal first exits from the explicit \(H_{L1}\) trigger law.}
Let
\[
H_{L1}=\{(q,w,u,\lambda)=(m-1,m-1,u,2):u\neq 2\}
\]
be the exact hole-family formula from the extracted defect classification, and let
\(
\rho=u_{\mathrm{source}}+1 \pmod m
\)
for the chosen source family. On phase-\(1\) current states of the candidate active orbit, one has
\[
q\equiv u-\rho+1 \pmod m.
\]
Therefore an alternate step lands in \(H_{L1}\) if and only if either
\[
(q,w,\Theta)=(m-1,m-2,1)\ \text{and}\ u\neq 2,
\]
or
\[
(q,w,\Theta)=(m-2,m-1,1)\ \text{and}\ u\neq 2.
\]
At the first state,
\(
 u=\rho-2
\), so the exit is allowed exactly when \(\rho\neq 4\), i.e. on the regular family; at the second,
\(
 u=\rho-3
\), so when \(\rho=4\) the exit is allowed and is the first one. Hence the family-dependent first exits are exactly
\[
T_{\mathrm{reg}}=(m-1,m-2,1),
\qquad
T_{\mathrm{exc}}=(m-2,m-1,1).
\]
As a consequence, the candidate orbit avoids all patched current classes before first exit, the actual branch agrees with the candidate orbit in the full current coordinates \((q,w,u,\Theta)\) up to first exit, and every nonterminal active current state is \(B\)-labeled.
