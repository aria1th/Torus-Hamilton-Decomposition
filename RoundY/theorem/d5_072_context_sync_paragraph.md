# D5 072 Context Sync Paragraph

The `072` results sharpen the current D5 frontier in one important way. We
should now treat the **coarse bridge** as essentially settled: the per-chain
marked-chain quotients do glue at the carry / distance-to-endpoint level.
Also, the **realization theorem** is now conditionally clear: if a deterministic
global quotient is exact for current `epsilon4`, then the short-corner detector
and the canonical clock descend automatically by corner-time. The live issue is
therefore no longer “does any bridge exist?” but “what is the smallest exact
bridge for current `epsilon4`?” The compute result says the bare `m`-state
bridge is too coarse, because it merges the early post-carry decorations, while
a small decorated bridge of the form `(\beta,a,b)` is supported on the checked
range. So from this point forward, the shared target should be the **decorated
exact bridge**, not the bare bridge, with symmetry/asymmetry and realization
questions interpreted relative to that refined object.
