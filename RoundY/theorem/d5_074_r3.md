## Executive answer

The post-`073` realization theorem should be stated at one level more abstractly
than before.

The right object is **not** the static local quotient `(beta,a,b)`. It is an
abstract deterministic quotient

\[
\pi:X\to Q
\]

of the theorem-side exact accessible object, where `Q` is whatever
**splice-compatible decorated bridge** ultimately survives.

At that level, the sharp conclusion is:

> If `Q` is deterministic and exact for current `epsilon4`, then the
> short-corner detector is visible on `Q` automatically.
>
> To recover the canonical clock, the extra hypothesis is **not** uniqueness of a
> short-corner state on each component. The needed extra input is only that each
> accessible forward orbit meets the descended short-corner set again.
>
> Under that recurrence hypothesis, the canonical clock descends uniquely as the
> residue modulo `m` of corner-time:
> \[
> \beta_Q(q)=T_{corner}(q)-2 \pmod m,
> \]
> where `T_corner(q)` means the residue class modulo `m` of any future hit time
> from `q` to the visible short-corner set.

So the key refinement beyond `073` is:

- **current `epsilon4` exactness is still the only event exactness needed**;
- **short-corner uniqueness on the quotient is not needed**;
- **if the final bridge is longer than one bare `m`-cycle, the normalization is
  still modulo `m`, not modulo the component length**.

## 1. Abstract setup

Let `X` be the theorem-side exact accessible object with deterministic successor
`F`, current event

\[
\epsilon_4:X\to E,
\]

phase

\[
\kappa:X\to \mathbb Z/m\mathbb Z,
\]

and canonical clock

\[
\beta_X:=-\kappa.
\]

Assume the accepted phase-corner package on `X`:

1. \(\kappa(Fx)=\kappa(x)+1\pmod m\);
2. the short corner is characterized by the two-step signature
   \[
   (\epsilon_4(x),\epsilon_4(Fx))=(\mathrm{flat},\mathrm{other}_{0010});
   \]
3. every short-corner state lies at phase \(\kappa=2\), equivalently at
   \(\beta_X=m-2\).

I am **not** assuming here that there is only one short-corner state globally.
The theorem below only uses the fact that every short-corner hit occurs at the
same phase residue.

Now let

\[
\pi:X\to Q
\]

be the candidate **splice-compatible decorated bridge**. For this note, that
means only:

- `Q` is a deterministic quotient with induced successor `G` satisfying
  \(\pi\circ F = G\circ\pi\);
- on regular non-splice stretches, `Q` refines the accepted decorated
  marked-chain picture;
- the splice / wrap step has been made deterministic on `Q`.

The word “splice-compatible” matters for the **construction** of `Q`, but the
realization theorem itself uses only the deterministic quotient structure once
that construction is in hand.

Assume the one exactness hypothesis

\[
\epsilon_4 = \bar\epsilon_4\circ \pi
\]

for some event readout \(\bar\epsilon_4:Q\to E\).

This is the exact same hypothesis singled out in `070`: current-event exactness,
and nothing stronger.

## 2. Visible short corner on the abstract bridge

Define the theorem-side short-corner set

\[
S_X:=\{x\in X:(\epsilon_4(x),\epsilon_4(Fx))=(\mathrm{flat},\mathrm{other}_{0010})\}.
\]

Define on `Q`

\[
S_Q:=\{q\in Q:(\bar\epsilon_4(q),\bar\epsilon_4(Gq))=(\mathrm{flat},\mathrm{other}_{0010})\}.
\]

### Lemma 2.1 (short-corner detector descends)

Under determinism and current `epsilon4` exactness,

\[
S_X = \pi^{-1}(S_Q).
\]

Equivalently, the short-corner detector on `X` factors through `Q`.

#### Proof

For any `x`, determinism gives

\[
\pi(Fx)=G\pi(x).
\]

Since \(\epsilon_4=\bar\epsilon_4\circ\pi\), we get

\[
(\epsilon_4(x),\epsilon_4(Fx))
=
(\bar\epsilon_4(\pi(x)),\bar\epsilon_4(G\pi(x))).
\]

So `x` is short-corner exactly when `pi(x)` satisfies the displayed condition
for `S_Q`. ∎

### Remark 2.2

This is the precise sense in which **current-event exactness is enough**.
No separate hypothesis is needed for:

- two-step exactness,
- future-word exactness,
- short-corner exactness,
- prior exactness of the canonical clock.

All future `epsilon4` words descend automatically once the quotient is
deterministic and current `epsilon4` is exact.

## 3. The correct corner-time theorem after `073`

The `073` note still phrased the componentwise theorem using a unique visible
short-corner state on each component. That is sufficient, but it is stronger
than necessary and is no longer the right default once the final bridge may be
longer than one bare `m`-cycle or may keep extra splice memory.

The sharper theorem uses the **set** `S_Q`, not a singleton corner state.

For `x in X` and `q in Q`, define future-hit sets

\[
H_X(x):=\{r\ge 0 : F^r x\in S_X\},
\]

\[
H_Q(q):=\{r\ge 0 : G^r q\in S_Q\}.
\]

### Theorem 3.1 (mod-`m` corner-time realization on the exact bridge)

Assume:

1. `Q` is a deterministic splice-compatible decorated quotient of `X`;
2. current `epsilon4` is exact on `Q`;
3. every accessible state has at least one future short-corner hit; this may
   be stated upstairs as \(H_X(x)\neq\varnothing\) for accessible `x`, or
   downstairs as \(H_Q(q)\neq\varnothing\) for accessible `q`, because
   \(S_X=\pi^{-1}(S_Q)\).

Then:

1. for every accessible `q`, \(H_Q(q)\neq\varnothing\);
2. every two elements of \(H_Q(q)\) are congruent modulo `m`;
3. therefore the residue class
   \[
   T_Q(q):=[r]_m \in \mathbb Z/m\mathbb Z \qquad (r\in H_Q(q))
   \]
   is well defined;
4. `T_Q` satisfies
   \[
   T_Q(Gq)=T_Q(q)-1 \pmod m;
   \]
5. the canonical clock descends uniquely by
   \[
   \beta_Q(q):=T_Q(q)-2 \pmod m;
   \]
6. this descended clock satisfies
   \[
   \beta_Q\circ\pi = \beta_X,
   \qquad
   \beta_Q(Gq)=\beta_Q(q)-1 \pmod m.
   \]

#### Proof

Fix accessible `q`, and choose any accessible preimage `x` with \(\pi(x)=q\).

Because \(S_X=\pi^{-1}(S_Q)\), we have

\[
H_Q(q)=H_X(x).
\]

So by assumption \(H_Q(q)\neq\varnothing\).

Now let \(r\in H_Q(q)\). Then \(F^r x\in S_X\), hence `F^r x` is a short-corner
state. By the phase-corner package, every short-corner state has

\[
\beta_X(F^r x)=m-2.
\]

Since `beta_X` drifts by `-1`,

\[
\beta_X(F^r x)=\beta_X(x)-r \pmod m.
\]

Therefore

\[
\beta_X(x)-r \equiv m-2 \pmod m,
\]

so

\[
r \equiv \beta_X(x)+2 \pmod m.
\]

Thus every `r` in \(H_Q(q)\) has the same residue modulo `m`, proving (2) and
hence the well-definedness of \(T_Q\).

For the drift identity, note that

\[
r\in H_Q(Gq)
\iff
G^r(Gq)\in S_Q
\iff
G^{r+1}q\in S_Q
\iff
r+1\in H_Q(q).
\]

So the hit-time residue shifts by `-1 mod m`, which gives

\[
T_Q(Gq)=T_Q(q)-1 \pmod m.
\]

Now define \(\beta_Q:=T_Q-2\). For `q=pi(x)`, any \(r\in H_Q(q)\) satisfies

\[
\beta_X(x)\equiv r-2 \equiv T_Q(q)-2 = \beta_Q(q) \pmod m,
\]

so \(\beta_Q\circ\pi = \beta_X\). The drift law for \(\beta_Q\) follows
immediately from that for `T_Q`.

Uniqueness on the accessible image of `pi` is immediate from
\(\beta_Q\circ\pi=\beta_X\). ∎

### Corollary 3.2 (how to read the theorem when the bridge is longer than one `m`-cycle)

If an accessible component of `Q` has length larger than `m`, the descended
clock is **still modulo `m`**.

The correct normalization is not by the component length. It is by the residue
class modulo `m` of future hitting times to `S_Q`.

So the right formula is always

\[
\beta_Q = T_Q - 2 \pmod m,
\]

with `T_Q` understood as the **mod-`m` corner-time residue**, not as a clock
modulo the full component length.

This is the key normalization correction to the `073` component-length phrasing.

## 4. Stronger window form and relation to `070`

The theorem above is the sharp form needed for the post-`073` bridge, because it
makes no singleton-corner assumption and no single-`m`-cycle assumption.

There is also a stronger but more concrete version that recovers the `070`
formula verbatim.

### Corollary 4.1 (window form)

In addition to the hypotheses of Theorem 3.1, assume every accessible state of
`Q` hits `S_Q` within the next `m` steps.

Then for each `q` there is a unique

\[
\tau_Q(q)\in\{0,1,\dots,m-1\}
\]

with

\[
G^{\tau_Q(q)}q\in S_Q,
\]

and

\[
\beta_Q(q)=\tau_Q(q)-2 \pmod m.
\]

#### Why uniqueness in the window is automatic

By Theorem 3.1, all future hits of `S_Q` from a fixed `q` are congruent modulo
`m`. So there can be at most one such hit in `\{0,1,\dots,m-1\}`.

### Interpretation

This is exactly the earlier `070` corner-time picture. The point is that the
stronger “next hit within one `m`-window” hypothesis is a **convenient special
case**, not the minimal hypothesis needed for canonical-clock descent.

## 5. Answers to the `074` subquestions

### 5.1 What is the minimal exactness hypothesis?

The sharp natural answer is:

> **current `epsilon4` exactness on a deterministic quotient**.

That is the only event exactness used by the proof.

No extra assumption is needed for:

- two-step short-corner exactness,
- future `epsilon4`-word exactness,
- exactness of a visible `beta` coordinate,
- exactness of a candidate `(beta,a,b)` coordinate,
- exactness of the canonical clock itself.

I am **not** claiming a formal converse against every possible ad hoc weakening.
What I am claiming is the sharper theorem-level fact relevant here:

- once current `epsilon4` is exact, nothing stronger is needed for the descent
  argument;
- if current `epsilon4` is not exact, this route has no reason to identify the
  short corner on the quotient.

So for the present D5 route, current `epsilon4` exactness is the right exactness
hypothesis.

### 5.2 What componentwise / periodicity hypothesis is needed?

Not singleton short-corner structure.

The minimal extra dynamical hypothesis is only:

> every accessible forward orbit meets the visible short-corner set `S_Q` again.

Equivalently, every accessible state has at least one future hit to `S_Q`.

A stronger and easier-to-use special case is:

> every accessible state hits `S_Q` within the next `m` steps.

That stronger window form lets one use an actual least future hit
\(\tau_Q(q)\in\{0,\dots,m-1\}\), but it is not the minimal condition needed for
mod-`m` descent.

### 5.3 Is short-corner uniqueness automatic once the bridge is exact?

No.

Exactness for current `epsilon4` makes the detector **visible** as the subset
`S_Q`, but it does **not** force `S_Q` to be a singleton on each component.

A final bridge may keep extra splice memory or future-decoration memory after the
corner. Then several distinct quotient states can all satisfy the same visible
short-corner signature.

What is automatic is weaker and exactly what the theorem uses:

- every hit to `S_Q` from a fixed state lies in one residue class modulo `m`;
- therefore the canonical mod-`m` clock is still forced.

So “one visible corner state per component” is a convenient special case, not a
necessary hypothesis.

### 5.4 How should the theorem be normalized if the final bridge is longer than one bare `m`-cycle?

Normalize **modulo `m`**, not modulo the full component length.

The descended canonical clock is always

\[
\beta_Q = T_Q - 2 \pmod m,
\]

where `T_Q` is the residue class modulo `m` of future hit times to `S_Q`.

If the component is longer than `m`, then one of two things can happen:

- `S_Q` appears only once on the component; then the component length is
  automatically a multiple of `m`, and the same formula still uses residue mod
  `m`;
- `S_Q` appears several times on the component; then successive hits differ by
  multiples of `m`, and again only the residue mod `m` matters for the
  canonical clock.

So the component length is secondary bookkeeping. The intrinsic clock modulus is
still `m`.

## 7. What still depends on the unresolved bridge construction?

The unresolved bridge branch still has to supply the following concrete facts.

### 7.1 The final state space

The final bridge may be a refinement or recoding of the local `(beta,a,b)`
picture. The realization theorem should therefore be read against an abstract
state `q in Q`, not against a hard-coded static chain type.

### 7.2 Determinism across the splice

The theorem does **not** apply to a quotient that is only exact on individual
chains. The bridge must already be a genuine deterministic quotient on the full
accessible exact object, including the splice step.

This is exactly why the static chain-type `(beta,a,b)` model from `073` is not
already enough.

### 7.3 Exactness for current `epsilon4`

The final decorated state must make the current event a function of the quotient
state. This is the exactness hypothesis that triggers the whole realization
argument.

### 7.4 A visible recurrence statement for the short corner

For the sharp theorem above, one still needs either:

- the minimal recurrence statement that every accessible orbit meets `S_Q`, or
- the stronger window statement that every accessible state reaches `S_Q`
  within the next `m` steps.

This is a statement about the actual accessible component structure of the final
bridge. It is not settled merely by the local per-chain decorated rule.

### 7.5 The explicit visible form of `S_Q`

In the naive local `(beta,a,b)` model, the short corner is the local state

\[
(\beta,a,b)=(m-2,0,1).
\]

For the final splice-compatible bridge, the correct statement is more careful:

- the short-corner set is **always** the event-defined set `S_Q`;
- it may project to the local pattern `(m-2,0,1)` on regular interior states;
- it need not be literally one quotient state unless the final bridge proves no
  further split survives there.

So the final bridge construction may change the visible coordinate expression of
`S_Q`, but not the event-defined realization theorem.

## 8. Final theorem statement to carry forward

Here is the theorem I would now treat as the right Researcher-3 target.

### Realization theorem on the splice-compatible exact bridge

Let `X` be the theorem-side exact accessible D5 object with deterministic
successor `F`, current event `epsilon4`, and canonical clock
\(\beta_X=-\kappa\in\mathbb Z/m\mathbb Z\). Let `Q` be a deterministic
splice-compatible decorated quotient of `X` with successor `G`, and assume only
that current `epsilon4` factors through `Q`.

Define the visible short-corner set on `Q` by

\[
S_Q=
\{q:(\bar\epsilon_4(q),\bar\epsilon_4(Gq))=(\mathrm{flat},\mathrm{other}_{0010})\}.
\]

If every accessible state has a future hit to `S_Q`, then the set of all such
future hit times from a fixed state has a unique residue modulo `m`, and the
canonical clock descends uniquely by

\[
\beta_Q(q)=T_Q(q)-2 \pmod m,
\]

where `T_Q(q)` is that corner-time residue.

If, moreover, every accessible state hits `S_Q` within the next `m` steps, then
`T_Q(q)` is represented by the unique integer `tau_Q(q) in {0,...,m-1}` with
`G^{tau_Q(q)}q in S_Q`, so the formula becomes

\[
\beta_Q(q)=\tau_Q(q)-2 \pmod m.
\]

### Practical meaning

So the realization burden after `073` is genuinely small.

Researcher 3 no longer needs:

- a guessed explicit clock,
- a guessed singleton corner state,
- or a guessed final decorated coordinate.

Researcher 3 only needs the bridge branch to supply a deterministic exact
quotient for current `epsilon4` together with the short-corner recurrence
property. After that, the canonical clock is forced.
