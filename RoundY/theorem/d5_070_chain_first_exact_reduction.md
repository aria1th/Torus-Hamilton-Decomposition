# D5 070 Chain-First Exact Reduction

This note gives a stable descriptive home for the exact-reduction result
recorded in:

- `theorem/d5_070.md`

## Statement

The right first exact reduction object is a marked length-`m` chain, not yet a
full cycle.

The correct logical order is:

1. extract the deterministic marked chain;
2. build the exact quotient induced on that chain;
3. only then ask whether the quotient promotes to a cycle.

## Why this matters

This is the note that first makes the chain-vs-cycle distinction explicit in a
theorem-shaped way. It is the clean precursor to the later bridge/globalization
notes.
