# D5 072 Conditional Realization Theorem

This note gives a stable descriptive home for the realization statement
originally recorded in:

- `theorem/d5_072_r3.md`

## Statement

If the global bridge quotient is deterministic and exact for current
`epsilon4`, then:

- the short-corner detector descends automatically;
- the canonical clock descends by corner-time.

So the realization problem is not:

> invent a clock.

It is:

> build the quotient on which current `epsilon4` is exact.
