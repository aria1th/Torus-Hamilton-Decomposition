## Subdirectories

### `autonomous/`

Contains 25 autonomous perturbation notes (`d5_autonomous_perturbation_note_v1.md`
through `v25.md`).  These are intermediate results from an automated
perturbation-search loop run in Phase 2–4, testing carry-swap and local-flag
variants.  They are not part of the theorem chain; use them only if
reconstructing the automated search history.

### `distribution/`

Contains session-exchange bundles (`.tar.gz` + unpacked directories) that were
passed between the human researcher and Codex agents.  Named by date and
update stage (e.g., `roundy_d5_codex_job_bundle_20260312_update_048`).  The
canonical mathematical content has already been extracted into `theorem/`,
`checks/`, and the session notes above.

### `theorem/`

Promoted theorem notes — the canonical D5 chain.  See
`theorem/d5_082_frontier_and_theorem_map.md` for the reading order.

### `checks/`

Stable compute-evidence JSON files supporting the theorem chain.

### `specs/`

Codex-facing work templates and compute-request specifications.
