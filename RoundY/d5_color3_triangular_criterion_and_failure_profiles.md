Candidate lemmas:
- [P] First-return reduction:
  Since S increases by +1 under every f_c-step,
  f_c is a Hamilton cycle on the whole torus iff R_c is a single cycle on P_0.

- [P] Clean-frame detection:
  A clean frame for R_c exists iff there is a nonzero order-m vector k with
    R_c(x+k)=R_c(x)+k
  for all x.
  Proof sketch:
  * If A R_c A^{-1}(b,z)=(B(b), z+beta(b)), then k=A^{-1}(0,0,0,1) works.
  * Conversely, if such k exists, choose a complement W to <k> and write x=w+zk.
    Then R_c preserves the quotient by <k> and acts by fiber translation.

- [P] Triangular skew-product Hamiltonicity criterion:
  Let
    R(b,s,z) = (B_0(b,s), s+1, z + beta(b,s))
  on A x Z_m x Z_m, with A=(Z_m)^2.
  Let
    B(b,s) = (B_0(b,s), s+1)
  and
    U = B^m |_{s=0}.
  For each U-orbit O, let \tilde O be the lifted B-orbit and define the orbitwise monodromy
    M(O) = sum_{y in \tilde O} beta(y) mod m.
  Then:
  1. B-orbits are exactly the m-fold lifts of U-orbits.
  2. Over one lifted B-orbit of length L, the skew product splits into gcd(m,M(O))
     cycles, each of length L*m / gcd(m,M(O)).
  3. Hence R is a single cycle iff U is a single cycle and M(O) is a unit mod m
     on that unique U-orbit.

- [P] Color-3 instantiation:
  In relative coordinates
    (q,w,v,u) = (x_4,x_0,x_1,x_2)
  on P_0, the current witness gives
    R_3(q,w,v,u) = (B_3(q,w,u), v + beta_3(q,w,u)),
  with
    B_3(q,w,u) = ( q + 1_{(w,u)=(1,0)},  w+1,  u + gamma_3(q,w) ),
  where
    gamma_3(q,w)=1  iff
      (w=0 and q in {0,4}) or (w=2 and q!=4) or (w=4 and q=3).
  Therefore w is a strict clock coordinate.
  The section return
    U_3 = B_3^m |_{w=0}
  is a map on (q,u) in (Z_m)^2.
  For odd m, U_3 is a single m^2-cycle and the unique orbit has monodromy
    M_3 = -1 mod m,
  so R_3 is a single m^4-cycle, and f_3 is Hamilton on the full torus.
