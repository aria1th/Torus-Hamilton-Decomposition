# D4 Layer-2 Gauge Classification 01

## Goal

This bundle addresses Task ID `D4-LAYER2-GAUGE-CLASSIFICATION-01` from
`RoundX/codex_job_request_3.md`.

The target was the reduced layer-2 family on

`H = {S = 2, q = 0}`, with `q = x0 + x2 mod m`,

where the rule is allowed to depend only on the two bits

- `u = 1_{x0 != 0}`
- `v = 1_{x3 = 0}`

and each of the four truth-table cells chooses one element of the Klein-four
subgroup generated by the slot swaps `(1 3)` and `(0 2)`.

Outside that family:

- `S = 0` is fixed to the affine-split pair
  - `q = 0 -> (3,2,1,0)`
  - `q != 0 -> (1,0,3,2)`
- `S = 1` is canonical
- `S >= 3` is canonical

## Main Classification Result

The reduced family has exactly `4` Hamiltonian rules on the full tested range

`m = 3,4,5,...,20`.

The surviving truth-table codes are:

- `[0,1,2,3]`
- `[1,0,3,2]`
- `[2,3,0,1]`
- `[3,2,1,0]`

in the pattern order

`[(u,v)=(0,0), (1,0), (0,1), (1,1)]`.

Interpreting `0,1,2,3` as

- `0 = none`
- `1 = odd swap`
- `2 = even swap`
- `3 = both swaps`

these four survivors are exactly one Klein-four orbit:

- current witness
- current witness XOR global odd swap
- current witness XOR global even swap
- current witness XOR global odd+even swap

No additional Hamiltonian rule appears in the reduced family on `m = 3..20`.

## Current Witness Up To Gauge

The current witness is the code

`[0,1,2,3]`,

that is:

- `(u,v)=(0,0)` -> canonical
- `(u,v)=(1,0)` -> odd swap
- `(u,v)=(0,1)` -> even swap
- `(u,v)=(1,1)` -> both swaps

So its layer-2 patch is exactly the two-gate rule:

- odd gate on `x0 != 0`
- even gate on `x3 = 0`

This gives layer-2 support size

`m^2 - m + 1`.

The reduced-family search therefore supports the interpretation that the
meaningful rigid object is the differential two-gate pattern, not the literal
four-row table.

## Line-Union Gauge

The line-union gauge is the global odd-toggle of the current witness:

`[1,0,3,2]`.

Equivalently, on `H = {S = 2, q = 0}`:

- odd swap on `x0 = 0`
- even swap on `x3 = 0`
- both when both tests hold
- canonical otherwise

So its layer-2 defect support is the union

`{x0 = 0} U {x3 = 0}`,

with size

`2m - 1`.

This is the smallest support among the four surviving gauge representatives:

- current witness: `m^2 - m + 1`
- line-union gauge: `2m - 1`
- even global gauge: `m^2 - 1`
- odd+even global gauge: `m^2 - m + 1`

## Verified Line-Union Return Formulas

Using

`phi(a,b,q) = (a, b, q-a, -q-b)`

to parametrize `P_0`, the line-union gauge satisfies the following verified
first-return formulas on `P_0`:

```text
R_0(a,b,q) =
  (a-1, b,   q-1)   if q = 0
  (a-2, b+1, q-1)   if q = m-1 and b = 1
  (a-1, b+1, q-1)   otherwise

R_1(a,b,q) =
  (a,   b-1, q+1)   if q = 0
  (a+1, b-2, q+1)   if q = m-1 and a = m-1
  (a+1, b-1, q+1)   otherwise

R_2(a,b,q) =
  (a,   b+1, q-1)   if q = 0
  (a+1, b,   q-1)   if q = m-1 and b = 2
  (a,   b,   q-1)   otherwise

R_3(a,b,q) =
  (a+1, b,   q+1)   if q = 0
  (a,   b+1, q+1)   if q = m-1 and a = 0
  (a,   b,   q+1)   otherwise
```

On `Q = {q = m-1}`, the second-return maps are:

```text
T_0(a,b) = (a - 1_{b=1},         b - 1)
T_1(a,b) = (a - 1,               b - 1_{a=m-1})
T_2(a,b) = (a + 1_{b=2},         b + 1)
T_3(a,b) = (a + 1,               b + 1_{a=0})
```

The corresponding affine odometer conjugacies are:

```text
T_0: (u,v) = (1-b,   -a)
T_1: (u,v) = (-a-1,  -b)
T_2: (u,v) = (b-2,   a)
T_3: (u,v) = (a,     b)
```

with standard odometer

`O(u,v) = (u+1, v+1_{u=0})`.

In particular, `T_3` for the line-union gauge is already the standard odometer
in the `(a,b)` coordinates.

## Verification Range

The script checked, for every `m = 3..20`:

- full Hamiltonicity of all `256` reduced-family rules until first failure;
- exact equality of the line-union `R_c` formulas with the actual first-return
  maps;
- exact equality of the line-union `T_c` formulas with the actual second-return
  maps;
- affine conjugacy of each line-union `T_c` to the standard odometer;
- single-cycle behavior of each line-union `T_c` on `Q`.

Every line-union check passed on the full tested range.
