# RouteY-Existence

This directory is the curated extraction of
`tmp/d5_consolidated_support_archive_2026-03-28_v3.tar`.

It is a **parallel existence-oriented layer** for the resonant `d=5` program.
It does **not** replace the canonical current D5 frontier in
[`RoundY`](../RoundY/README.md).

Use the three layers this way:

- `RoundY/`: canonical current D5 proof frontier and manuscript-order packet
- `RouteY-Existence/`: existence / seam-surgery reading of the resonant
  residual program
- `tmp/d5_consolidated_support_archive_2026-03-28_v3.tar`: exhaustive raw
  archive with older bundles, exploratory material, and extra reports

## What Is Promoted Here

This layer keeps only the proof-carrying spine of the archive:

- `status/`: archive status docs
- `manuscript/`: the resonant manuscript snapshot
- `theorem/`: existence notes and generic-late arithmetic chain
- `checks/`: short exact reports for the main generic-late arithmetic steps
- `scripts/`: replay scripts corresponding to those checks
- `reference/`: the structural `+5` carrier/splice note

It intentionally omits the prior tar bundles, exploratory superseded notes,
and larger archive duplicates.

## Mathematical Role

The promoted reading here is:

1. local visibility is treated as closed baseline input;
2. the generic-late good branch is reduced to an explicit `M`-point reduced
   base machine after the `7`-split;
3. this reduced machine is not best read as one global odometer;
4. the right object is an explicit **odometer-with-seams** or
   **finite seam-surgery** permutation;
5. the remaining open core in this lane is therefore a global cycle theorem
   for that explicit seam-glued reduced machine.

So this directory is not a second complete D5 proof. It is a stable parallel
lane for the existence-style and seam-surgery reading of the resonant
residual problem.
